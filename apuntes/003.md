Operadores
===

Los operadores aritmeticos son:
---
```py
suma + 
eg: a + b
resta -
eg: a - b
multiplicacion *
eg: a * b
division /
eg: a / b
vidision entera //
eg: a // b
potencia **
eg: a ** b
modulus %
eg: a % b
```

Una forma alternativa de imprimir texto con formato es usando la instruccion f asi:  
```py

a = 3
b = 2
suma = a + b
""" print("el resultado de la suma es", suma) """
print(f'El resultado de la suma es {suma} ')

```

Los operadores de asignación son:
---

```py
asignación = 
eg: a = 3
```

Los operadores de comparacion:
---

```py
igualdad ==
diferencia != 
mayor que  >
menor que  <
mayor o igual que >=
menor o igual que <=

```
