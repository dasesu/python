Tipos de datos
===
En python tenemos los siguientes tipos
numeric -> intger, numero complejo, float
dictionary:
boolean:
set:
sequence type -> Strings, List, tuple

Si quiero conocer el tipo de dato que tiene una variable puedo usar la instruccion type(x) asi

```py
x=10
print(x)
print(type(x))
```

Python admite que le indiquemos una pista al programador de que tipo de variable se trata, lo hacemos asi

```py
x: str= "Hola mundo"
print(x)
print(type(x))
```

Para definir variables segun el tipo deseado asignamos valores que ese tipo de dato por ejemplo
```py
x: int= 9
print(x)
print(type(x))

x: float= 9.5
print(x)
print(type(x))

x: bool= True
print(x)
print(type(x))

x: str= "Hola Mundo"
print(x)
print(type(x))

```

Python es un lenguaje de programación orientado a objetos y por lo tanto son clases las que van a estar almacenando esta información

Manejo de cadenas
---
# cadena ( string )
Para concatenar cadenas usamos el simbolo de '+'
```py
miGrupoFavorito = "Aerosmith"

print("Mi grupo favorito es " + miGrupoFavorito)

miGrupoFavorito = "Aerosmith" + "the best rock band"

print("Mi grupo favorito es " + miGrupoFavorito)

```

Cuando lo que queremos es imprimir por pantalla podemos concatener el formato, que no es lo mismo a unir 2 cadenas de caracteres, esto podemos hacerlo usando el simbolo de coma asi:

```py
miGrupoFavorito = "Aerosmith"

print("Mi grupo favorito es ", miGrupoFavorito)

```
