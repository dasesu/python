Variables
===

Debemos saber que python es un lenguaje debilmente tipado, es decir no debemos indicarle el tipo de dato. tambien es un lenguaje interpretado, por lo que podemos cambiar el tipo de dato sobre la marcha.

```py
miVariable = 2
print( miVariable )
print( miVariable )
print( miVariable )

miVariable = 10
print( miVariable )
```

Es buena practica quitar el espacio entre los parentesis y el nombre de la variable, quedando asi
```py
miVariable = 2
print(miVariable)
print(miVariable)
print(miVariable)

miVariable = 10
print(miVariable)
```

Para conocer la ubicación en memoria de una variable podemos usar la instrucción id()
```py
x = 10
y = 2
z = x + y
print(x)
print(y)
print(z)
id(x)
print(id(x)) // Esto retorna la direccion de memoria de x
```
